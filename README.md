# 🏠 House Price Prediction using Machine Learning

## 📌 Overview

This project predicts house prices using machine learning techniques based on features such as location, size, and other housing attributes.

## ⚙️ Technologies

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib / Seaborn
* Jupyter Notebook

## 📊 Models Used

* Linear Regression
* Random Forest
* Support Vector Regression
* XGBoost

## 📈 Results

* XGBoost achieved the best performance
* Models evaluated using MAE, RMSE, and R²

## 🧹 Data Processing

* Data cleaning
* Feature engineering
* Handling missing values

## 🚀 How to Run

1. Download the notebook
2. Open in Jupyter Notebook
3. Run all cells
